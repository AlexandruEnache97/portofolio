Shipwrecked Island RPG

    This game is still in development.

    The prinicipal idea of the game is that you are shipwrecked on some unknown tropical island. You can pick up items, interact with objects to collect stuffs, craft basic tools and objects.

    The UI is still in development, for the moment I implement the level of the player which increase by collecting objects and pick them up on the ground.

    The quest system will guide the player through surviving.

    The items collected or crafted are stored in the inventory panel.

    The crafting recipe is accesible only if the all items are found in the inventory. You can increase the numbers of crafting recipes if you got the items for it.
